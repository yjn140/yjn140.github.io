没有男朋友的陈瑭，也没有女朋友。她身边的位子就那么空着。陈瑭大概也觉得无聊，挤到前排两个女孩儿身边坐下，和她们分着翻几张娱乐周刊，不时评价几句。”真是个不爱学习的肤浅家伙”，荣荣在心里严厉地批评陈瑭，觉得她不应该是自己的菜。最近他真是对自己失望透顶了。 陈瑭笑起来眯着眼睛，清澈得就像一滴露珠。女孩们很快把报纸都翻完了，陷人无话可讲的沉默。“下一刻，也许她会转头落寞地看着窗外。”荣荣推测。电视剧里不都是这么演的吗?什么“你的笑是你的保护色”，放下快乐的伪装沉静下来，就该缠绵于自己的不幸了。荣荣很想捕捉到陈瑭的忧郁，虽然这忧郁也不会因他而起，但会使陈瑭更接近一个完整的人。 然而她却也回到了之前的位子上，闭上眼睛打起了盹，头靠在车窗上随车子摇摇晃晃。蝶翼般的长睫毛微微翕动，没有皱眉，没有叹气，嘴角甚至带着刚才欢笑的余韵。 荣荣完败，人家根本没有不快乐。 也有人，感情是不外露的吧，比如他自己。荣荣忍不住又编织起了另一种可能，却被司机想了下嘲叭打断了。这一声遭到小区沿街住户的谩骂，却居然没有吵醒陈瑭。她是真的睡着了呀，看来困得不行了。荣荣走过去用手指轻轻点了点她的肩膀，“嘿，下车了。”她的眼睛硬撑着张开，眼珠转了一圈又阖上眼皮，瞌睡帝，真是可爱得要命。荣荣又推了她一把，“到家啦。”陈瑭这才勉强站起来，她揉揉眼睛，直直看了一眼荣荣，痛苦得简直要哭出来。 就是这么，她两手死死抓住荣荣的衬衫袖子，踉踉跄跄下了车。 这就算认识了？不，其实他认识她很久了。 那是十一月的某天中午，阳光跳进车里，橙色流转，整个世界都是亮堂堂的。 只要有耐心，他的祈祷都会实现的。就像他有条不紊地实施学习计划，最后都会换来好成绩。 上下学时，荣荣已经能堂堂地坐在陈瑭身边跟她作伴了。下车后一起从小区门口到楼底下也是自然而然的事。他进24栋，她穿过花园走进26栋。荣荣没有提出过要送她到她家楼下，他认为不应该操之过急一下子要求太多，慢慢的进程让他感到踏实而不是做梦。 陈瑭话不多，就爱笑，不知道哪儿有那么多的事可乐的。荣荣天生就口拙，现在感到压力很大，如果无话可讲，会不会让陈瑭觉得自己很无趣呢？于是他就用哼哼歌来代替沉默。跟苍蝇蚊子叫似的，兴许陈瑭笑的就是这个，想到自己的傻帽，荣荣也一乐。女孩儿说的最多的话就是“季海荣你走慢点儿”，然后小小碎步子赶上来，倒让荣荣怀疑自己是在有心欺负她哪。 下晚学的时候，荣荣总是感到饿。但他是个有自制力的人，从小养成了晚上八点之后就不吃东西的习惯。他还打算中年以后要修炼过午不食。陈瑭就不行了，小区门口炸鸡店飘出的油香在叫她的魂呢。天气渐渐转冷了，她又不肯穿多，饥寒交迫挺可怜见儿的。荣荣就买一袋鸡排给她，她能兴高采烈地啃一路，一点儿也不为身材发胖担心。搜到一半，用牙签戳一块，冲他笑笑，“饿死我了，你要不要来一口？”荣荣撇撇嘴，“垃圾食品！回家记得刷牙，我看你牙缝里都是。”糖糖轻轻“哼”一声，乐滋滋把独食吃得满嘴油光。荣荣不用鼻子哼哼他的小调了，改用口哨吹，这样感觉上轻快些。 对了，糖糖是荣荣给取的小名，他为自己的创意感到很满意，糖糖，甜甜的。 真是和谐得一塌糊涂。 （三） 可这一天不可避免地一定会到来的，不是么？陈瑭还是有了新的男朋友。 荣荣在校车上坐定，莫名其妙乐呵呵地把书包搁在靠窗的位置给陈瑭占着座，凝神找那小曲的调儿，还没来得及哼起来，他就从大巴的前挡风玻璃里看到一个高高瘦瘦穿校服打领带的男生把她送上了校车。陈瑭习惯成自然地在荣荣身边落了座，顺着他的目光看去，没明白他在看什么，于是用手肘蹭蹭，“看什么呢？”车子开出校门转了弯，荣荣才把盯着那男生的目光收回来。没什么了不起，他心想，你也早晚是浮云。 这个男生
